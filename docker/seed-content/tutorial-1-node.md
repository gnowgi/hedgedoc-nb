# Lesson 1 · A concept

Everything in a knowledge graph starts with a **concept** — a single idea, drawn
as a **node**.

In nodeBook you write a concept with `#`, and put *what kind of thing it is* in
square brackets:

```nodeBook
# Dog [Animal]
```

That's it — one line, one node. `Dog` is the concept; `[Animal]` is its **type**
(what kind of thing it is).

### Try it
Edit the block above (or make a new ` ```nodeBook ` block) and add a concept of
your own:

- `# Cat [Animal]`
- `# Oak [Plant]`
- `# Paris [City]`

Each line you add becomes another node. A node on its own isn't very interesting
yet — the magic starts when we **connect** concepts.
